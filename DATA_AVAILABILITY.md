# Data Availability

## MIMIC-III and MIMIC-IV

MIMIC data are available from PhysioNet to credentialed users:

- [MIMIC-III](https://physionet.org/content/mimiciii/)
- [MIMIC-IV](https://physionet.org/content/mimiciv/)

Access requires completion of required training and a data-use agreement. This repository does **not** include MIMIC raw tables, patient identifiers, or derived patient-level feature exports.

## Local feature tables (not redistributed)

Full reproduction requires locally prepared files placed by the user. Column and sheet requirements are documented in [docs/preprocessing_excel.md](docs/preprocessing_excel.md).

| File | Purpose |
|------|---------|
| `data/mimic_dataset.xlsx` | MIMIC-III / MIMIC-IV pre-aggregated feature tables (user-prepared; entry point for the public pipeline) |
| `data/mimic_t0_labs.parquet` | Generated by `scripts/prepare_t0_labs_parquet.py` (Excel proxy) or SQL export |
| `data/mimic_severity_scores.parquet` | SOFA proxy from Excel or full SOFA/APACHE/SAPS from SQL (optional) |

Validate your Excel before running:

```bash
python scripts/validate_excel_input.py
python scripts/run_locked_pipeline.py
```

## Chinese external validation (not redistributed)

Geographic validation used institution-held cohorts. If reproducing F3/NH evaluation locally, place authorized files under:

- `data/external/mimic_validation_cohorts.xlsx`
- `data/external/f3_rdw_correction.xls`
- `data/external/nh_relabel_candidates.csv`

These files are not included in the public repository.

## What we do publish

- HTTPS research demonstration calculator: https://k.mixaihub.top (not for clinical use)
- Code, SQL scripts, model card, methods documentation
- Released 8-feature model weights for research inference (`model_weights/8_features_model.pkl`)
- Non-patient configuration such as the LASSO feature name list (`config/low_k_lasso_feature.csv`)
